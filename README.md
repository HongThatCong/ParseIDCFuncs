# ParseIDCFuncs
Parse IDCFuncs (ext_idcfunc_t struct in expr.hpp in IDA SDK) in ida.dll/ida64.dll of IDA 7.x

Create ext_idcfunc_t struct for every IDC function, name and comment prototype of that function.
